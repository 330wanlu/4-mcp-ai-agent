# 费用报销管理制度

> 文档编号：FIN-EXP-003  
> 版本：V4.1  
> 生效日期：2025-02-15  
> 适用范围：全体员工费用报销  
> 归口部门：财务部

## 1. 基本原则

1.1 报销须「先发生、后报销」，禁止无业务实质的费用。  
1.2 票据真实、完整、合法；电子发票须可查验。  
1.3 单笔报销须在费用发生之日起 **60 日内** 提交，逾期须总监说明原因。

## 2. 报销科目（常用）

| 科目代码 | 名称 | 说明 |
|----------|------|------|
| EXP-01 | 差旅交通 | 机票/火车/网约车等，须关联出差单号 |
| EXP-02 | 差旅住宿 | 须关联出差单号 |
| EXP-03 | 差旅补助 | 系统按出差单自动计算，一般无需贴票 |
| EXP-04 | 业务招待 | 餐饮/礼品；须填写招待对象与事由 |
| EXP-05 | 办公耗材 | 非固定资产类办公用品 |
| EXP-06 | 培训发展 | 外部培训、考试费（须培训申请单） |
| EXP-07 | 通讯补贴 | 按职级月度定额，超额自理 |

## 3. 票据要求

3.1 增值税专用发票：须公司抬头、税号齐全；备注栏尽量填写业务事由。  
3.2 普通电子发票：上传 PDF/OFD 原件，禁止仅截图。  
3.3 出租车发票：须注明起终点；网约车须行程单 + 电子发票一致。  
3.4 **招待费发票不得拆分多张规避审批**。

## 4. 审批权限

| 单笔金额 | 审批 |
|----------|------|
| ＜ 1000 元 | 直属主管 |
| 1000–4999 元 | 直属主管 + 部门负责人 |
| ≥ 5000 元 | 上述 + 财务经理；≥ 20000 元须分管副总 |

## 5. 差旅相关报销规则

5.1 差旅类科目（EXP-01/02/03）**必须绑定有效出差申请单**，否则财务退回。  
5.2 差旅标准以《差旅管理制度》为准；本制度只规定票据与审批流程。  
5.3 超差旅标准部分：无事先批准的，超标金额员工自理。

## 6. 业务招待与差旅边界

6.1 出差期间邀请客户用餐：记 **业务招待（EXP-04）**，不记差旅补助餐补，当天差旅补助按半额计。  
6.2 个人独自用餐：不开发票报销，适用差旅补助。  
6.3 礼品费单次超过 500 元须合规部备案。

## 7. 禁止报销清单（摘录）

- 罚款、违约金  
- 个人会员费、健身卡  
- 无审批的家电、数码设备（走固定资产流程）  
- 酒精饮品（商务宴请啤酒/葡萄酒单次合计 ≤ 300 元除外）

## 8. 发放周期

每周三批次支付已审结报销；遇节假日顺延。
