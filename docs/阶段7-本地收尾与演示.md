## 3. 遇到的问题与处理

### 3.1 README 仍停在阶段 4

- **处理**：按 0–7 重写启动、端口、Demo 与联验命令，并链到 architecture / demo-script。

### 3.2 「一键」在 Windows 上不宜单进程阻塞

- **处理**：`start_local.ps1` 用 `Start-Process` 各开窗口跑 API 与 Vite，避免同 shell 卡死。

### 3.3 全量 pytest 很慢

- **处理**：`smoke_phase7.ps1 -SkipPytest` 可只跑 deps + demo；完整验收再去 `-SkipPytest`。

### 3.4 需要你帮忙的

**无硬依赖。** 演示时保持 Postgres / Memurai Running，并配置 `ARK_API_KEY`。  
可选：`.env` 设 `AUTH_PROVIDER=dev_header` 以便 Console 用户名生效。
